# Gecachte jaarstukken-PDF’s (Cuatro)

Deze map bevat de **volledige** jaarstukken-PDF’s waarmee `scripts/build_taakveld_lasten_staat_2020_2025.py` de tabel vult (2022–2024). Ze zijn te groot om zonder meer in git te zetten; haal ze zo nodig op met de `curl`-regels in `../taakveld-lasten-staat-2020-2025.md`.

De korte bestanden `Jaarstukken_YYYY.pdf` **één map hoger** zijn in deze werkmap **geen** bruikbare vervanging (1 pagina, vrij zonder tekstlaag).
