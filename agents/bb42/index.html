Wat doet BB42?

Agent BB42 is het geautomatiseerde zoek- en rapportagedeel van jullie keten: periodiek trefwoorden uitzetten tegen openbare registers (o.a. KOOP / lokale regelgeving), output als JSON. Geen automatische wijziging van de site — review, GO Yvonne, daarna handmatig in de data en deploy.

Werkwijze

  • python run.py op de VPS (zie README.md)
  • Rapport bekijken (hieronder of bestand bb42-last.json)
  • GO Yvonne → opname in beleidsnota-per-taakveld-data.js → ACC → Ricardo → PROD

Laatste rapport bekijken

Kies het JSON-bestand van een run, of plak de JSON-output en klik Toon.
