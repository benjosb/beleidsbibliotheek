Productie is read-only

  • Live is een alleen-lezen weergave — geen inhoud bewerken op de server; wijzigingen via git → ACC → promote.
  • In de praktiek wekelijks of bij een release — niet ad hoc op de productieserver.
  • De mutatielijst (opvolging reacties): aanpassen in het bronbestand in de repo en mee deployen; op productie geen beheer-knop.
