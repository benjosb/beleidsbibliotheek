VOSviewer – Wassenaar verkiezingsprogramma's 2026
================================================

VOSviewer Online (app.vosviewer.com) accepteert alleen JSON:
1. Run: python3 maak_vosviewer_json.py
2. Open vosviewer_wassenaar_2026.json in https://app.vosviewer.com

VOSviewer Desktop (tekst → map maken):
1. Create → Create a map based on text data
2. Directory: deze map
3. Thesaurus: thesauris.txt

Bestanden:
- 1__VVD.txt t/m 9__Volt.txt = programmatekst per partij
- thesauris.txt = term-vervangingen
- vosviewer_wassenaar_2026.json = klaar voor VOSviewer Online
